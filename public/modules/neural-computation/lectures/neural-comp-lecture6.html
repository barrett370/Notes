Numerical Issues with Softmax

Neural networks are usually implemented with fixed length representations of real numbers.
Obviously, these representations are finite representations of a uncountably infinite set R\RR

For instance, the maximal value of the float64 data type in NumPy can represent is ≈1.8×10308\approx 1.8\times10^{308}≈1.8×10308
When computing the numerator in ezjLQ\frac{e^{z_j^L}}{Q}QezjL​​ we can easily exceed this limit
