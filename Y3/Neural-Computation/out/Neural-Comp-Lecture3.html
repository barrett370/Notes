The Learning Task, TTT

  • Instead of deterministically predicting an output yyy for a given input xxx we will now train a probabilistic model represented by a conditional density function
(6)

Pmodel(y∣x;θ)P_{model}(y | x ; \theta)Pmodel​(y∣x;θ)

  • Where:

    • y←y\leftarrowy← density function of output
    • x←x\leftarrowx← input
    • θ←\theta \leftarrowθ← parameter of model

  • Given training data and a family of probability models we need to choose the parameter(s) θ\thetaθ which are most appropriate for the data. We call this the Maximum likelihood estimate

  • Given independent training data (x1,y1),⋯ ,(xn,yn)(x_1,y_1), \cdots, (x_n,y_n)(x1​,y1​),⋯,(xn​,yn​) and a probabilistic model PmodelP_{model}Pmodel​ with parameter θ\thetaθ, the likelihood function is defined as:

  • L(θ;...)\mathcal{L}(\theta; ...)L(θ;...) is the likelihood that the observed data came from the model with parameter θ\thetaθ

Maximum Likelihood Estimate (MLE)

  • Given training data and a family of models indexed by the parameter θ\thetaθ, which of the models are most likely to have produce the data?
