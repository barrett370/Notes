Lecture 1

Regression

  • The task of finding the relationship (mathematical function) between one or more numerical inputs (independent variables) and one or more numerical outputs (dependent variables)

  • Curve fitting

    • Given a set of points, try ot learn a function to describe them
    • Given a value xxx, we can predict the corresponding value yyy
    • Not just for straight line fitting

Simple example

Let us consider a simple linear example with 1 independent variable, xxx & 1 dependent variable, yyy

(1)

D={(x1,y1),...,(xn,yn)}={(xi,yi)}i=1ND = \{(x_1,y_1),...,(x_n,y_n)\} = \{(x_i,y_i)\}_{i=1}^ND={(x1​,y1​),...,(xn​,yn​)}={(xi​,yi​)}i=1N​

Model the relationship between xxx and yyy with the function f(w,x)f(\textbf{w},x)f(w,x), s.t yi≈f(w,xi)y_i \approx f(\textbf{w},x_i)yi​≈f(w,xi​)
Measurements of yyy, subject ot noise are defined by,

(2)

yi=f(w,w)+ϵy_i = f(\textbf{w},w) + \epsilonyi​=f(w,w)+ϵ

Where ϵ\epsilonϵ is a random number drawn from some continuous probability density function.
Goal is to find some w\textbf{w}w that solves, or provides the best approximation to the above equation.

First, let us approach this as a optimisation problem in which the objective is to find the value of w (denoted w*) that minimises some loss or objective function L(w)L(\textbf{w})L(w)

(3)

w∗=arg min⁡wL(w)\textbf{w}* = \argmin_w L(\textbf{w})w∗=wargmin​L(w)

Intuitively, L(w)L(\textbf{w})L(w) should be designed to capture the difference between the data and the predictions of the model, and seek to minimise this.
One common choice for L(w)L(\textbf{w})L(w) is least-squares error. Given our dataset DDD and modelling function f(w,x)f(\textbf{w},x)f(w,x), we construct ∀d∈D\forall d \in D∀d∈D a residual error defined as:

(4)

ri(w)=yi−f(w,xi)r_i(\textbf{w}) = y_i - f(\textbf{w}, x_i)ri​(w)=yi​−f(w,xi​)

The least squares error (LSE) loss function is defined in terms of residuals as:

(5)

LLSE(w)=∑i=1Nri2=rTrL_{LSE}(\textbf{w}) = \sum_{i=1}^N r_i^2 = \textbf{r}^T\textbf{r}LLSE​(w)=i=1∑N​ri2​=rTr

It is important to note that the above definition implicitly has no upper bound but is restricted to being strictly posive, thus, allowing us to find a minimum value.

(6)

w∗=arg min⁡wLLSE(w)\textbf{w}^* = \argmin_\textbf{w} L_{LSE}(\textbf{w})w∗=wargmin​LLSE​(w)

Optimisation is a very difficult problem to solve and so, for now, we resitct ourselves to a specifically linear case.

N.B. when we say linear we do not reference the problem being of the form "y=mx+cy = mx + cy=mx+c" but instead that our problem is linear in it's unknown parameters

Linear models take the form:

(7)

f(w,x)=w0ϕ0(x)+⋯+wM−1ϕM−1(x)=∑i=0M−1wiϕi(x)f(\textbf{w},x) = w_0\phi_0(x) + \cdots + w_{M-1}\phi_{M-1}(x) = \sum_{i=0}^{M-1} w_i\phi_i(x)f(w,x)=w0​ϕ0​(x)+⋯+wM−1​ϕM−1​(x)=i=0∑M−1​wi​ϕi​(x)

Our function is a linear combination of a set of basis functions, {ϕi(x)}i=0M−1\{\phi_i(x) \}_{i=0}^{M-1}{ϕi​(x)}i=0M−1​ weighted by our free parameters {wi}i=0M−1\{w_i\}_{i=0}^{M-1}{wi​}i=0M−1​.

A common choice of basis function is the polynomials {xi}i=0M−1\{x^i\}_{i=0}^{M-1}{xi}i=0M−1​

For a finite set of data points DDD we can re-write the equation for f(w,x)f(\textbf{w},x)f(w,x) in matrix form by defining a matrix Φ\PhiΦ with components Φij=ϕj(xi)\Phi_{ij} = \phi_j(x_i)Φij​=ϕj​(xi​) which yields the equation:

(8)

f(w)=Φwf(w) = \Phi wf(w)=Φw

Where the variable xxx is swallowed by the construction of Φ\PhiΦ in fff

We can now begin to solve our equation here. The residuals defined here can be written as:

(10)

r=y−Φwr = y - \Phi wr=y−Φw

And our loss function, becomes:

(11)

LLSE(w)=(y−Φw)T(y−Φw)L_{LSE}(\textbf{w}) = (\textbf{y}-\boldsymbol{\Phi}\textbf{w})^T(\textbf{y}-\boldsymbol{\Phi}\textbf{w})LLSE​(w)=(y−Φw)T(y−Φw)

As we have observed, LLSEL_{LSE}LLSE​ has no upper bound but does have a lower bound. To minimise LLSEL_{LSE}LLSE​ we find the point at which LLSE′=0L_{LSE}' = 0LLSE′​=0 where LLSE′L_{LSE}'LLSE′​ is it's first derivative with respect to it's free parameters.

If we differentiate LLSEL_{LSE}LLSE​ with respect to w\textbf{w}w we get:
