Linear Dependence

Let v1⃗,v2⃗…,vM⃗∈RN\vec{v_1},\vec{v_2}\ldots,\vec{v_M}\in \R^Nv1​​,v2​​…,vM​​∈RN

Let the set {vecv1,v2⃗,…,vM⃗}\left\{ \\vec{v_1},\vec{v_2},\ldots, \vec{v_M}\right\}{vecv1​,v2​​,…,vM​​} is linearly dependent if there are scalars λ1,λ2,…,λM\lambda_1,\lambda_2,\ldots,\lambda_Mλ1​,λ2​,…,λM​ not all of which are zero such that:
λ1v1⃗+λ2v2⃗+…λMvM⃗=0\lambda_1\vec{v_1} + \lambda_2\vec{v_2} + \ldots \lambda_M\vec{v_M} = 0λ1​v1​​+λ2​v2​​+…λM​vM​​=0

The set {v1⃗,v2⃗,…,vM⃗}\left\{ \vec{v_1},\vec{v_2},\ldots, \vec{v_M}\right\}{v1​​,v2​​,…,vM​​} is linearly independent if it is not linearly dependent

However for a basis set of vectors we want more than them to just be linearly independent, we want all vectors in the set to be orthogonal to each other.

Demonstrating linear dependence/ independence

Given a set of linear simultaneous equations you can show their linear dependence/ independence via the process of Gaussian Elimination.

For a matrix V\bf VV with columns v1⃗,…,vM⃗\vec{v_1},\ldots,\vec{v_M}v1​​,…,vM​​, if V\bf VV can be reduced to a diagonal matrix with non-zero elements then the set of equations are linearly independent, otherwise they are linearly dependent.

Interpretation

The set {v1⃗,v2⃗,…,vM⃗}\left\{ \vec{v_1},\vec{v_2},\ldots,\vec{v_M}\right\}{v1​​,v2​​,…,vM​​} is linearly dependent if at least one member of the set can be written as a linear combination of the others

Vector Set Bases

A basis for a vector space is a coordinate system. In R3\R^3R3 we in general use the x,y,zx,y,zx,y,z coordinate system, Any vector:

v⃗=[vxvyvz]∈R3\vec{v} = \begin{bmatrix} v_x \\ v_y \\ v_z \end{bmatrix} \in \R^3v=⎣⎢⎡​vx​vy​vz​​⎦⎥⎤​∈R3

can be written:

v⃗=vx×ux⃗+vy×uy⃗+vz×uz⃗\vec{v} = v_x \times \vec{u_x} + v_y \times \vec{u_y} + v_z \times \vec{u_z}v=vx​×ux​​+vy​×uy​​+vz​×uz​​

where ux⃗,uy⃗\vec{u_x},\vec{u_y}ux​​,uy​​ and uz⃗\vec{u_z}uz​​ are unit vectors

ux⃗=[100],uy⃗=[010],uz⃗=[001]\vec{u_x} = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} , \vec{u_y} = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} , \vec{u_z} = \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix}ux​​=⎣⎢⎡​100​⎦⎥⎤​,uy​​=⎣⎢⎡​010​⎦⎥⎤​,uz​​=⎣⎢⎡​001​⎦⎥⎤​

Here ux⃗,uy⃗\vec{u_x},\vec{u_y}ux​​,uy​​ and uz⃗\vec{u_z}uz​​ are unit vectors as:

∥ux⃗∥=∥uv⃗∥=∥uz⃗∥=1\|\vec{u_x}\| = \|\vec{u_v}\| = \|\vec{u_z}\| = 1∥ux​​∥=∥uv​​∥=∥uz​​∥=1

and are orthogonal as:

ux⃗⋅uy⃗=ux⃗⋅uz⃗=uy⃗⋅uz⃗=0\vec{u_x} \cdot \vec{u_y} = \vec{u_x} \cdot \vec{u_z} = \vec{u_y} \cdot \vec{u_z} = 0ux​​⋅uy​​=ux​​⋅uz​​=uy​​⋅uz​​=0

Any set of 3 orthogonal unit vector can be used as a coordinate system in R3\R^3R3, such as set is called a basis for R3\R^3R3 or more strictly an orthonormal basis

Definition of a basis of RN\R^NRN

A basis for the vector space RN\R^NRN is the set of vectors u⃗1,u⃗2,…,u⃗N\vec u_1, \vec u_2, \ldots, \vec u_Nu1​,u2​,…,uN​ s.t. they are all unit vectors and mutually orthogonal.
