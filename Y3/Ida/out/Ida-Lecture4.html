Document Similarity

Suppose ddd is a document and qqq si a query

  • if ddd contains the same words in the same proportions, then vec(d)vec(d)vec(d) and vec(q)vec(q)vec(q) will point in the same direction
  • If ddd and qqq contain different words then vec(d)vec(d)vec(d) and vec(q)vec(q)vec(q) will point in different directions
  • Intuitively, the greater the angle between vec(d)vec(d)vec(d) and vec(q)vec(q)vec(q), the less similar ddd and qqq are.
